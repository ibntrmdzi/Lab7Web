Buka aplikasi Visual Studio dan buat file baru di folder lab7_php_dasar dengan nama php-dasar. Kemudian masukan kode tersebut.
```php
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>PHP Dasar</title>
</head>
<body>
  <h1>Belajar PHP Dasar</h1>
  <?php
    echo "Hello World";
  ?>
</body>
</html>
```
